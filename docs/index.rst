PyGuitarPro
===========

PyGuitarPro is a package to read, write and manipulate GP3, GP4 and GP5 files.
Initially PyGuitarPro is a Python port of AlphaTab which is a HaXe port of
TuxGuitar.

To anyone wanting to create their own the best guitar tablature editor in
Python this package will be the good thing to start with.
