Installation
============

Install the extension with:

.. code-block:: console

    $ pip install pyguitarpro

or using ``easy_install``:

.. code-block:: console

    $ easy_install pyguitarpro

or manually:

.. code-block:: console

    $ hg clone https://bitbucket.org/Perlence/pyguitarpro
    $ cd pyguitarpro
    $ python setup.py install
